## プライバシーポリシー 本アプリは、ユーザーのプライバシーを尊重し、必要最小限のデータのみを扱います。 ### 1. 取り扱う情報 本アプリは、ユーザー操作に応じて次の情報を扱います。 - チャット内容 - 設定情報（モデル、システムプロンプト、温度など） - 添付ファイル情報（画像・テキスト抽出結果など） - APIキー（設定内容に応じて `sessionStorage` または `localStorage`） ### 2. 保存場所 データは主に以下へ保存されます。 - ブラウザ `localStorage` - ブラウザ `sessionStorage` - ユーザーが連携した Google Drive（任意） 開発者管理のサーバーへ会話データを保存する設計ではありません。 ### 3. 外部送信 ユーザー入力および必要なコンテキストは、応答生成のため外部APIへ送信される場合があります。 - Gemini API - OpenAI API 送信先サービスでのデータ処理は、各サービスのポリシーに従います。 ### 4. APIキーの扱い APIキーの保存方式は設定に依存します。 - 既定: `sessionStorage`（ブラウザセッション終了で削除） - 保存オプションON時: `localStorage`（ユーザーが削除するまで保持） ### 5. ユーザー責任 本アプリはサーバーレス設計です。 データ保護、端末保護、認証情報管理はユーザー自身の責任で行ってください。 ### 6. 改定 本ポリシーは、必要に応じて予告なく更新される場合があります。 --- ## Privacy Policy This app respects user privacy and handles only the minimum data needed for operation. ### 1. Information Processed Depending on user actions, the app may process: - Chat content - Settings (model, system prompt, temperature, etc.) - Attachment-related data (images, extracted text, etc.) - API keys (`sessionStorage` or `localStorage`, based on settings) ### 2. Storage Locations Data is primarily stored in: - Browser `localStorage` - Browser `sessionStorage` - User-linked Google Drive (optional) The app is not designed to store chat data on developer-controlled servers. ### 3. External Transmission User input and required context may be sent to external APIs for response generation: - Gemini API - OpenAI API Data handling by those services is governed by their own policies. ### 4. API Key Handling API key storage depends on user settings: - Default: `sessionStorage` (cleared when the browser session ends) - When "remember" is enabled: `localStorage` (kept until removed by the user) ### 5. User Responsibility This app is serverless. You are responsible for data protection, device security, and credential management. ### 6. Policy Updates This policy may be updated without prior notice when needed.
