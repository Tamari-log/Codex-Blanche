## プライバシーポリシー 本アプリは、ユーザーのプライバシーを尊重し、最小限のデータ取り扱いを行います。 ### 1. 収集する情報 本アプリは、以下の情報をユーザーの操作に応じて扱います。 * チャット内容 * 設定情報（システムプロンプト、モデル設定など） * APIキー（セッション中のみ） これらの情報は、ユーザーの利用環境内に保存されます。 --- ### 2. データの保存場所 * ローカルストレージ（ブラウザ内） * セッションストレージ（APIキー） * Google Drive（ユーザーが連携した場合のみ） 開発者のサーバーにデータが送信・保存されることはありません。 --- ### 3. 外部送信 ユーザーが入力した内容は、以下の目的で外部APIに送信される場合があります。 * AIによる応答生成（Gemini / OpenAI など） 送信されたデータは、各サービス提供者のポリシーに従って処理されます。 --- ### 4. APIキーの扱い APIキーはセッション中のみブラウザ内に保存され、 ブラウザ終了時に削除されます。 --- ### 5. 利用者の責任 本アプリはサーバーを持たない設計のため、 データの管理・保護はユーザー自身の責任で行ってください。 --- 本ポリシーは必要に応じて変更されることがあります。 --- ## Privacy Policy This App respects user privacy and minimizes data handling as much as possible. ### 1. Information Handled The App may handle the following data based on user actions: * Chat content * Settings (system prompts, model configuration, etc.) * API keys (session-based only) All data is stored within the user's environment. --- ### 2. Data Storage Data may be stored in: * Browser local storage * Browser session storage (for API keys) * Google Drive (only if the user enables synchronization) No data is stored on any server controlled by the developer. --- ### 3. External Transmission User input may be sent to external APIs for processing, such as: * AI response generation (Gemini, OpenAI, etc.) These services process data according to their own privacy policies. --- ### 4. API Keys API keys are stored only in session storage and are cleared when the browser session ends. --- ### 5. User Responsibility Since this App does not provide server-side storage, users are fully responsible for managing and protecting their own data. --- This policy may be updated as needed.
