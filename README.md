# Codex-Blanche

自分のために作った、サーバーレスなAIクライアントです。  
GitHubに置いてあるのは、URLを使うためです。使いたい人はどうぞ。

🇺🇸 English version → ./README.en.md

---

## Live

👉 https://tamari-log.github.io/Codex-Blanche/

---

## 特徴

- サーバー不要（完全クライアントサイド）
- APIキーはローカル管理（sessionStorage）
- Google Drive連携で設定・履歴を同期
- モデル（Gemini / OpenAI）切り替え可能
- システムプロンプト・温度・コンテキストを細かく制御
- ログ機能あり（開発者向け）

---

## 設計思想

- アプリは「骨組み」、データはユーザーのもの  
- 囲い込みなし、ロックインなし  
- 自分で管理できる人間のためのツール  

一般向けのサービスではありません。

---

## セットアップ

1. APIキーを用意
   - Gemini または OpenAI

2. アプリを開く

3. 設定画面から入力
   - APIキー
   - モデル
   - 必要なら Google Client ID

4. （任意）Google Drive連携で同期

---

## 機能

### 基本機能

- AIチャット（Gemini / OpenAI）
  - プロバイダの切り替え
  - モデルの選択
  - 統一されたチャットUI

- システムプロンプト制御
  - カスタム指示の設定
  - プリセット（ペルソナ）の切り替え
  - 設定の保存と再利用

- 生成設定
  - 温度（temperature）の調整
  - 最大トークン数（コンテキスト長）の制御

---

### データ管理・保存

- ローカル状態管理
  - 会話履歴をブラウザに保存
  - プリセットや設定もローカル保持

- Google Drive同期
  - 複数端末間で履歴・設定を同期
  - ログイン時に自動同期
  - タイムスタンプベースの競合処理

- APIキーのセッション管理
  - APIキーは sessionStorage に保存
  - ブラウザ終了時に自動削除

---

### チャット機能

- メッセージ編集
  - 過去メッセージのインライン編集
  - 編集内容は即時保存

- 再生成
  - 任意の地点からAI応答を再生成

- セッション管理
  - 複数会話の管理
  - 名前変更・削除・ピン留め

---

### UI / UX

- シンプルなチャットUI
  - 会話に集中できる設計
  - モバイル対応

- ダーク / ライトモード
  - テーマ切り替え

- プリセットパネル
  - プリセットと履歴の切り替え
  - チャット履歴の検索

---

### 開発者向け機能

- ログビューア
  - consoleログ（log / warn / error）を収集
  - 詳細設定から閲覧可能

- エラーハンドリング
  - APIエラーの表示
  - 生成の中断（Abort対応）

- モジュール構造
  - UI / API / 同期処理の分離
  - 依存関係を明示した設計

---

### アーキテクチャ

- 完全クライアントサイド
  - サーバー不要

- データはユーザー所有
  - サーバーに保存しない
  - Google Driveを利用した任意同期

- シンプルな同期モデル
  - タイムスタンプによる競合解決
  - 削除フラグ（tombstone）による整合性維持

---

## 注意

- APIキーはブラウザのセッションに保存されます  
- セキュリティは自己責任です  
- 同期はシンプルな設計のため、並行編集には弱い場合があります
